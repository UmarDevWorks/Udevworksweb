My Projects

H&T Clock

A humidity and temperature display with NTP time and date.

This project involves an ESP8266/ESP32 microcontroller, a DHT11 temperature and humidity sensor, and an SSD1306 OLED display. The microcontroller fetches data from the DHT11 sensor and displays temperature, humidity, time, and date on the OLED display.

Full code:

Code available on GitHub

Wiring Connections:

ESP8266/ESP32 to OLED SSD1306:

  • SDA & SCL Pins: Connect to the corresponding pins on the OLED display.

DHT11 Sensor:

  • Data Pin: Connect to a GPIO pin on the microcontroller.

Code Summary:

  1. Initialization: Initializes the OLED, DHT11, and WiFi connections.
  2. Loop Function: Fetches data from DHT11, updates display with temperature, humidity, and time.
  3. Functionality: Manages data display and WiFi connection for time syncing.

Summary:

The ESP8266/ESP32 microcontroller connects to the OLED display and DHT11 sensor, displaying temperature, humidity, and synced time. Code handles data collection and display management. Proper wiring ensures functional communication between components.
